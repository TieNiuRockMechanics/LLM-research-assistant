# 基于正交各向异性介质理论的地应力地震预测方法.

## One-line Summary

本研究提出一种基于正交各向异性（OA）介质理论的地应力地震预测方法，推导出正交各向异性水平应力差异比（ODHSR）参数公式，以弥补传统HTI介质预测模型忽视页岩中VTI特征的不足。[Ma 2017, pp. 1-2]

## Research Question

如何在地震地应力预测中同时考虑页岩气储层的水平层状互层矿物（VTI 特征）和垂直裂缝扰动（HTI 特征）的影响，以建立更符合实际页岩储层的可压裂性评价参数？[Ma 2017, pp. 1-3]

## Study Area / Data

- **物理模型背景**：研究针对页岩气储层，该类储层通常具有超低孔低渗特征，并同时含有水平层状互层矿物（VTI）和定向排列的垂直或近垂直裂缝（HTI）。[Ma 2017, pp. 2-5]
- **验证数据**：选取了某页岩储层工区的测井数据（Well A）对提出的公式和方法进行验证。测井曲线包括弹性参数和各向异性参数。[Ma 2017, pp. 6-9]

## Methods

### 1. 正交各向异性（OA）介质应力应变关系推导
基于胡克定律的一般形式，利用正交各向异性介质的刚度矩阵和柔度矩阵之间的关系，推导了OA介质的应力和应变关系 [Ma 2017, pp. 3-5]。

### 2. 水平地应力及各向异性参数公式推导
假设地下岩石是有界的且不能移动，存在垂直方向的主应力和两个水平方向的应力，且水平方向上应变为零。在此条件下，利用OA介质柔度系数推导出了最小水平应力（σ_x）、最大水平应力（σ_y）的表达式，并与垂直地应力（由密度测井曲线积分估算）建立联系 [Ma 2017, pp. 3-5]。

在此基础上，定义了正交各向异性水平应力差异比（ODHSR），并推导出其用OA介质弹性刚度常数（c_{ij}）表示的精确公式（公式 12），以及通过各向异性参数近似表示的近似公式（公式 32）。ODHSR 是最大和最小水平应力变化差与最大水平应力的比值。[Ma 2017, pp. 3-6]

### 3. 公式验证
通过两种方式进行验证：[Ma 2017, pp. 5-9]
- **退化验证**：将本文推导的 ODHSR 精确公式退化为 HTI 介质下的 DHSR 公式，并与 Gray 推导的 DHSR 计算结果进行曲线对比，验证了公式的准确性。
- **实际数据验证**：利用实际工区测井数据（Well A）计算 ODHSR 和 DHSR 曲线并对比。结果表明，ODHSR 与 DHSR 曲线趋势基本一致，但在高值部分略高，差异归因于 VTI 各向异性参数的影响。

## Key Findings

1.  **新参数定义**：在正交各向异性（OA）介质理论框架下，本研究定义了“正交各向异性水平应力差异比”（ODHSR），该参数同时考虑了垂向（VTI）和水平（HTI）对称轴对地应力的影响，更符合实际页岩储层特性。[Ma 2017, pp. 5-9]

2.  **公式推导与实现**：成功推导出 ODHSR 的计算公式（精确式和近似式），实现了利用叠前方位地震反演获取的弹性参数和各向异性参数计算连续 ODHSR 剖面的地震地应力预测方法。[Ma 2017, pp. 3-6]

3.  **公式合理性证实**：
    - 本文精确公式退化得到的 HTI 介质 DHSR 与 Gray 的 DHSR 曲线基本一致，证明了退化公式的正确性。[Ma 2017, pp. 6-9]
    - 实际数据计算的 ODHSR 曲线与 DHSR 曲线趋势基本一致。高值部分的差异经分析归因于 VTI 各向异性参数的影响，当VTI参数γ设为零时，两者一致，证明了 ODHSR 公式的合理性。[Ma 2017, pp. 6-9]

4.  **工程应用潜力**：ODHSR 低值表明该区域最小和最大水平应力接近，易于压裂成网，有利于储层中油气的运移和开采。[Ma 2017, pp. 3-9]

## Limitations

1.  **适用构造条件**：ODHSR 公式是在水平两个方向应变为零的假设条件下推导得到的，因此适用于构造不复杂的页岩储层，未考虑复杂构造应力。[Ma 2017, pp. 6-9]
2.  **公式复杂度**：文中指出 ODHSR 精确公式过于复杂，需要进一步研究用正交各向异性介质各向异性参数表征的近似公式。[Ma 2017, pp. 6-9]
3.  **地应力预测精度**：该方法的后续工作有待研究构造应力对地应力的影响，以提高地应力预测精度。[Ma 2017, pp. 6-9]
4.  **验证数据**：验证过程使用井数据（Well A）进行，未展示基于实际地震数据的二维/三维预测剖面效果。论文中提及的地震预测方法实现，未从索引片段中确认具体的叠前方位地震反演细节。

## Assumptions / Conditions

1.  **介质模型**：页岩储层被等效为正交各向异性（OA）介质，同时考虑 VTI 和 HTI 特征。[Ma 2017, pp. 1-2]
2.  **边界条件**：假设地下岩石是有界的，不能移动，因此水平方向上的应变为零。[Ma 2017, pp. 3-5]
3.  **应力假设**：假设存在垂直方向的主应力和两个水平方向的应力。[Ma 2017, pp. 3-5]
4.  **适用场景**：推导出的公式适用于构造不复杂的页岩储层。[Ma 2017, pp. 6-9]
5.  **数据获取**：ODHSR 的地震预测方法建立在叠前方位地震反演能有效获取弹性及各项异性参数的基础上。[Ma 2017, pp. 5-6]

## Key Variables / Parameters

- **ODHSR (Orthorhombic Differential Horizontal Stress Ratio)**：正交各向异性水平应力差异比，核心评价参数，其值越低区域越易压裂成网。[Ma 2017, pp. 1-5]
- **σ_x, σ_y, σ_z (或 σ_v）**：分别表示最小水平地应力、最大水平地应力、垂直地应力。[Ma 2017, pp. 3-5]
- **c_{ij} (i,j=1..6)**：正交各向异性介质的弹性刚度常数，共有9个独立常数。[Ma 2017, pp. 3-5]
- **s_{ij}**：正交各向异性介质的柔度系数，弹性刚度矩阵的逆矩阵元素。[Ma 2017, pp. 3-5]
- **ε, δ, γ**：VTI 介质的 Thomsen 各向异性参数。[Ma 2017, pp. 5-6]
- **V_{P0}, V_{S0}**：垂直方向的纵、横波速度。[Ma 2017, pp. 5-6]
- **ρ**：密度。[Ma 2017, pp. 5-6]
- **λ, μ**：各向同性背景介质的拉梅参数。[Ma 2017, pp. 5-6]
- **Δ_N**：裂缝的法向弱度。[Ma 2017, pp. 5-6]
- **Z_N**：裂缝的法向柔度。[Ma 2017, pp. 5-6]

## Reusable Claims

1.  **Claim**：基于正交各向异性（OA）介质理论推导的水平应力差异比（ODHSR），相较于传统仅基于 HTI 介质的 DHSR，能够同时考虑页岩储层中水平层理（VTI）和垂直裂缝（HTI）的综合效应，因此更符合实际地质情况。
    - **Conditions**：适用于同时存在水平层理和垂直裂缝的页岩储层；假设构造不复杂且水平方向应变为零。[Ma 2017, pp. 3-9]

2.  **Claim**：通过将 ODHSR 公式退化为 HTI 介质下的 DHSR，并与 Gray 的 DHSR 模型进行对比，可以作为一种验证新模型正确性的有效方法。本文的退化结果显示两者曲线基本一致，存在的差异被归因于近似和累积误差。
    - **Conditions**：验证过程依赖于准确的弹性参数和各向异性参数输入；Gray 的 DHSR 模型作为基准。[Ma 2017, pp. 6-9]

3.  **Claim**：ODHSR 近似公式 (32) 中的高值异常，可通过引入 VTI 各向异性参数 γ 来解释。当将 γ 设为零时，ODHSR 近似公式结果与 DHSR 结果一致。
    - **Conditions**：基于线性滑动理论推导近似公式；用于解释实际数据或模型曲线中 ODHSR 与传统 DHSR 的差异来源。[Ma 2017, pp. 6-9]
