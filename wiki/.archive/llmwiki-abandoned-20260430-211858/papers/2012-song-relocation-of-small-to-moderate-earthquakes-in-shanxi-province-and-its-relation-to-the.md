# Relocation of Small to Moderate Earthquakes in Shanxi Province and Its Relation to the Seismogenic Structures.

## One-line Summary
利用1981-2008年模拟与数字地震台网数据和绝对/双差相对定位方法，对山西地区中小地震进行了精确重定位，结果显示震源分布与断陷盆地的地质构造具有清晰的空间相关性，且震源深度呈现北浅南深的特征 [Song 2012, pp. 1-2, 10-11]。

## Research Question
如何通过提高地震定位精度来更准确地刻画山西地震带的中小地震活动性及其与发震构造（断陷盆地、隆起区、断层）之间的关系 [Song 2012, pp. 2-3]。

## Study Area / Data
- **研究区**：山西及周边地区，核心为山西地震带（鄂尔多斯与华北地块交界处的山西地堑系）。
- **数据来源**：1981-2001年模拟观测台网和2002-2008年数字地震台网的震相数据（Pg、Sg等走时）[Song 2012, pp. 1-2]。
- **台站信息**：收集了山西地区10个台站下的地壳速度结构（部分台站仅有S波速度结构）[Song 2012, pp. 6-9字面意义推断]。其中提及山自皂（SZZ）、昔阳（XIY）、夏县（XAX）、右玉（YUY）、岢岚（KEL）等5个台站名称 [Song 2012, pp. 6-9]。
- **研究对象**：1981-2008年间山西地区ML≥1.0级地震 [Song 2012, pp. 6-9字面意义推断并结合图表描述]。
- **样本量**：重定位后震中水平误差≤5km的地震比例由65.8%提高到86.2%，7498次原始无震源深度的地震取得了深度结果 [Song 2012, pp. 1-2]。

## Methods
1.  **绝对定位（Absolute Location）**：
    -   **工具**：采用 Hypo2000 定位程序 [Song 2012, pp. 3-6]。
    -   **速度模型**：为应对山西地区地壳横向不均匀性（如太行山、吕梁山与沉积盆地并存），针对不同区域采用不同的地壳速度模型，并利用双同心圆加权过渡方法，在环形区域取相邻模型平均值，以逼近三维速度模型效果 [Song 2012, pp. 3-6]。
    -   **模型来源**：参考了山西及邻近地区已有的人工地震测深及接收函数等壳幔结构研究成果，并利用接收函数反演了部分台站的S波速度结构 [Song 2012, pp. 6-9]；同时参照了CRUST 2.0全球地壳模型 [Song 2012, pp. 3-6]。

2.  **相对定位（Relative Location）**：
    -   **方法**：在绝对定位基础上，选择地震集中的区域或有强震序列的区域，采用双差定位法（Double-Difference Method）进行相对定位 [Song 2012, pp. 3-6]。
    -   **输入**：本文直接使用震相报告的走时差，未采用波形互相关技术获取相对到时 [Song 2012, pp. 10-11]。

3.  **深度验证**：利用CAP（Cut and Paste）方法，对2003年洪洞地震进行近台宽频带波形拟合反演，以验证定位深度的可靠性。拟合显示深度在16-17 km之间，与定位结果（15 km）较为一致 [Song 2012, pp. 6-9位置推断]。

## Key Findings
1.  **精度提升**：重定位后，震中水平误差≤5km的地震比例从65.8%显著提升至86.2%；并为7498次原本无深度信息的地震赋予了震源深度，数据完整性大幅提高 [Song 2012, pp. 1-2]。
2.  **水平分布特征**：重定位后，地震总体分布格局与原始结果相比变化不大，绝大多数地震集中在中部断陷盆地带内，两侧隆起区较少，与区域地质构造的成带性相吻合。但重定位消除了原始目录的“棋盘格”状分布，使地震丛集性更高，呈条带状沿断层分布 [Song 2012, pp. 9-10, 10-11]。
3.  **深度分布特征**：震源深度整体呈现“北浅南深”的特征，存在由北向南逐渐加深的趋势，这与山西地堑由南向北逐渐撕裂、拉张的地质演化过程相吻合 [Song 2012, pp. 1-2, 9-10]。
4.  **构造刻画**：重定位结果可大致勾勒出各构造盆地的发震层下界，较清晰地分辨出断陷盆地、盆间隆起的位置，揭示了地震深度分布与盆山构造形态具有较好的相关性 [Song 2012, pp. 1-3]。
5.  **区域构造复杂性**：对2003年洪洞地震的震源机制反演显示该地震为NNE走向的左旋走滑事件，这表明以右旋走滑和拉张断层为主的山西裂谷带内仍存在左旋走滑断层，反映了该区域复杂的地震构造背景 [Song 2012, pp. 6-9]。
6.  **盆地具体表现**：不同盆地的深度特征得以显现。例如，大同盆地重定位后小震基本收缩在20km以上区域，基底向南掀斜的特征与深度分布一致 [Song 2012, pp. 9-10]。

## Limitations
1.  **速度模型限制**：尽管采用了分区加权一维速度模型，但山西地区位于地块交界带，地壳厚度和地形变化剧烈，一维模型仍无法精确描绘三维构造，结果精度受模型制约 [Song 2012, pp. 10-11]。
2.  **相对到时精度**：双差定位中直接使用震相报告到时差，而未采用波形互相关方法，走时差精度可能受损 [Song 2012, pp. 10-11]。
3.  **台站分布限制**：研究主要依赖山西本省台网，对省界附近的地震，台站偏向一侧（方位角分布不均）、台网布局不均匀，制约了定位精度 [Song 2012, pp. 10-11]。
4.  **方法覆盖范围**：双差相对定位仅应用于地震集中或发生过强震序列的子区域，并非对全部地震进行同等精度的约束 [Song 2012, pp. 3-6]。

## Assumptions / Conditions
- **假设**：采用的分区一维速度模型是地下真实速度结构的合理近似，且在双同心圆环形过渡区，速度结构的线性加权平均是有效的 [Song 2012, pp. 3-6]。
- **条件**：精确的构造解释依赖于地震定位的高度准确性和完备性 [Song 2012, pp. 2-3]。
- **条件**：CAP波形拟合反演的可靠性依赖于台站数量和分布，尽管文中提及方位角分布不佳，但认为仍满足方法要求 [Song 2012, pp. 6-9]。
- **条件**：关于发震构造与速度结构（如低速层）关系的推断，基于多震层深度与速度层深度的空间相关性 [Song 2012, pp. 9-10]。

## Key Variables / Parameters
- **震源参数**：震中位置（经度、纬度）、震源深度 [Song 2012, pp. 1-2]。
- **速度模型参数**：P波速度、S波速度、各台站下方分层速度结构 [Song 2012, pp. 6-9]。
- **误差指标**：水平定位误差（≤5km）、CAP波形拟合误差随深度变化曲线 [Song 2012, pp. 1-2, 6-9]。
- **地震参数**：矩震级（MW）、震源机制解（节面方位角、倾角、滑动角） [Song 2012, pp. 6-9]。
- **构造参数**：盆地沉积层厚度、发震层下界深度、断裂产状 [Song 2012, pp. 3-6, 10-11]。

## Reusable Claims
- **Claim 1**: 利用基于分区速度模型的绝对定位（Hypo2000）和双差相对定位的组合方法，可以显著提高构造复杂区域（如裂谷盆地与隆起相间地带）的中小地震目录的定位精度和深度信息完整性 [Song 2012, pp. 1-2]。**条件**：需要收集完备的近台走时资料。**限制**：结果精度仍受限于所用一维速度模型与真实三维结构的差异。
- **Claim 2**: 在山西裂谷带，中小地震的水平分布主要受断陷盆地和活动断裂控制，地震多成丛、成带分布，而与两侧隆起区关系较弱 [Song 2012, pp. 10-11]。**证据**：重定位后地震丛集性提高，条带分布与断层展布一致。
- **Claim 3**: 山西裂谷带震源深度的“北浅南深”趋势，可作为该地堑系由南向北逐渐拉张的地球动力学过程的现今活动性证据 [Song 2012, pp. 1-2, 9-10]。**证据**：地震深度剖面显示南部深度区间大于北部。
- **Claim 4**: 2003年山西洪洞发生的左旋走滑地震，表明该地区虽然整体处于右旋剪切拉张环境，但局部应力场复杂，存在不同性质的发震断层 [Song 2012, pp. 6-9]。
