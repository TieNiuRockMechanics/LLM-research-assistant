# Destruction of the North China Craton.

## One-line Summary
华北克拉通（NCC）在中生代-新生代经历了显著的岩石圈减薄、性质转变与地壳改造，太平洋板块俯冲是其破坏的主要动力学因素 [Zhu 2012, pp. 1-2]。

## Research Question
如何刻画华北克拉通破坏的时空分布、深部结构与浅部地质记录，并揭示克拉通破坏的机制与动力学过程？[Zhu 2012, pp. 1-2] 具体涉及破坏范围、岩石圈地幔在不同时代的性质与改造过程，以及板块俯冲等动力学触发因素。

## Study Area / Data
- **研究区**：华北克拉通（NCC）及邻区，包括东部地块、西部地块、中央造山带、太行山重力线、郯庐断裂带、苏鲁造山带等构造单元 [Zhu 2012, pp. 4-6, 8-9]。
- **地震观测数据**：华北克拉通内布设了间距约10–15 km的便携式宽频地震台，完成三条全长3400 km的宽角反射/折射剖面，并在渤海地区进行了海底地震仪与陆地便携式地震仪的联合调查，总长约930 km（见原文图1） [Zhu 2012, pp. 2-4]。利用这些数据获得了壳幔P波、S波速度扰动等三维层析成像模型 [Zhu 2012, pp. 6-6]。
- **地幔包体与捕虏晶样品**：来自华北克拉通主要构造块体和构造带的不同时代（古生代、中生代、新生代）的玄武岩、金伯利岩及基性岩中的橄榄岩包体、捕虏晶（如橄榄石、锆石） [Zhu 2012, pp. 6-8, 8-9]。
- **地球化学与同位素数据**：利用放射性同位素（Hf, Os）和非传统稳定同位素（Li, Mg, Fe）作为示踪剂进行实验研究 [Zhu 2012, pp. 6-8]。

## Methods
- **地震学方法**：
    - 基于波动方程的叠后深度偏移技术，对地震观测数据进行岩石圈-软流圈边界（LAB）成像 [Zhu 2012, pp. 2-4]。
    - **接收函数**（Receiver Functions）：利用S-to-P波接收函数绘制LAB深度，利用P-to-S波接收函数分析地壳-地幔边界（Moho）性质（如PpPs震相） [Zhu 2012, pp. 4-6]。
    - **地震层析成像**：反演体波走时数据，构建华北东部及邻区地幔深达700 km的P波速度（V_P）、S波速度（V_S）及波速比（V_P/V_S）异常三维模型 [Zhu 2012, pp. 6-6]。
- **岩石学与地球化学方法**：
    - 对地幔包体/捕虏晶进行**前人称**的矿物主量元素（如橄榄石Fo值、Mg#）及微量元素分析 [Zhu 2012, pp. 6-8]。
    - **地质年代学**与**同位素示踪**：利用Re-Os同位素体系确定岩石圈地幔的形成年龄和亏损地幔模式年龄，利用原位Li同位素及Mg-Fe同位素分析追踪地幔交代和改造过程 [Zhu 2012, pp. 9-10]。

## Key Findings
1.  **破坏的空间非均一性**：华北克拉通的岩石圈厚度和地壳结构存在显著的空间差异，这约束了破坏的范围 [Zhu 2012, pp. 1-2]。岩石圈厚度从东南部的60-70 km向西北加深至90-100 km，鄂尔多斯盆地之下为约200 km的厚岩石圈，而周边裂谷岩石圈较薄 [Zhu 2012, pp. 2-4]。
2.  **地壳的差异改造**：在东部华北克拉通（如太行山地区），接收函数的PpPs震相弥散且弱，表明存在由幔源岩浆底侵作用导致的厚壳幔过渡带；而在燕山地区，PpPs震相尖锐，反映了与克拉通破坏相关的下地壳快速拆沉导致的幔源物质与高层位地壳直接接触 [Zhu 2012, pp. 4-6]。
3.  **岩石圈地幔性质的时变特征**：
    - **古生代**：来自奥陶纪金伯利岩的橄榄岩包体和金刚石中的橄榄石包裹体具有高Fo值（>92 mol.%），显示太古宙克拉通型地幔特征，当时岩石圈厚度约200 km [Zhu 2012, pp. 2-2, 6-8]。
    - **中生代**：岩石圈地幔呈非均质性。东部胶东地区显示双层结构：上层为古老的高Mg#橄榄岩残留体，下层为晚白垩世以来的新增生地幔 [Zhu 2012, pp. 8-9]。
    - **新生代**：东部克拉通下的岩石圈地幔主体是新生的，具有饱满的、低Fo值（<91 mol.%）特征，类似于大洋岩石圈地幔，厚度减薄至80-120 km [Zhu 2012, pp. 2-2, 6-8, 9-10]。此外，晚中生代岩石圈地幔富水，而新生代岩石圈地幔极度贫水 [Zhu 2012, pp. 1-2]。
4.  **破坏的主导动力因素**：太平洋板块俯冲被认为是触发华北克拉通破坏的主要动力学因素 [Zhu 2012, pp. 1-2]。

## Limitations
- 岩石样品在空间上的分布有限，这制约了对岩石圈破坏范围和性质的认识 [Zhu 2012, pp. 2-4]。
- 华北克拉通岩石圈破坏的物理和化学过程仍在讨论中，存在多种机制（如拆沉、热-化学侵蚀、水弱化等），其主控作用尚需进一步研究 [Zhu 2012, pp. 4-6]。
- 基于深源捕掳体的研究可能受限于其采样地点和侵位时代，对不同区域和时间的代表性存在不确定性，如苏鲁带的下地壳麻粒岩包体年龄存在两种截然不同的解释 [Zhu 2012, pp. 8-9]。

## Assumptions / Conditions
- 将克拉通失去稳定性的地质现象定义为“克拉通破坏”，认为岩石圈减薄是其表层现象，克拉通破坏才是控制演化的根本 [Zhu 2012, pp. 2-2]。
- 地幔捕虏体和捕虏晶的组成及年龄能够代表其在被岩浆捕获时的岩石圈地幔状态。例如，利用橄榄石的Fo值可近似估算岩石圈地幔的形成年龄（太古宙地幔Fo > 92 mol.%，显生宙地幔Fo < 91 mol.%）[Zhu 2012, pp. 6-8]。
- 接收函数中PpPs震相的清晰度可以反映壳幔边界（Moho）的结构特征，尖锐震相对应尖锐界面，弥散震相对应厚过渡带 [Zhu 2012, pp. 4-6]。
- 地震层析成像速度异常可以反映地幔的温度、成分和部分熔融等状态，用于判别深部动力学过程 [Zhu 2012, pp. 6-6]。

## Key Variables / Parameters
- **岩石圈厚度**：从LAB深度推断，在华北东部从东南部的60-70 km变化到西北部的90-100 km，鄂尔多斯盆地约200 km [Zhu 2012, pp. 2-4]。
- **橄榄石Fo值**：利用摩尔百分比（mol.%）划分太古宙（>92）与显生宙（<91）岩石圈地幔 [Zhu 2012, pp. 6-8]。
- **地幔包体矿物同位素组成**：Re-Os模式年龄用于确定地幔形成与改造的时代；Li、Mg、Fe同位素组成的变化用于追踪地幔-熔体反应过程 [Zhu 2012, pp. 9-10]。
- **地震波速异常**：深部地幔的V_P, V_S, V_P/V_S比值，用于反演地幔结构 [Zhu 2012, pp. 6-6]。
- **接收函数PpPs震相（振幅）**：用于表征壳幔边界结构的尖锐或弥散程度 [Zhu 2012, pp. 4-6]。

## Reusable Claims
1.  **华北克拉通破坏的空间限定**：主要的岩石圈减薄和地壳改造集中发生在东部华北克拉通，而西部鄂尔多斯块体仍保留着约200 km厚的岩石圈根；东部LAB从东南（60-70 km）向西北（90-100 km）加深 [Zhu 2012, pp. 2-4]。**证据**：高密度地震台阵的LAB深度成像图 [Zhu 2012, pp. 2-4]。
2.  **新生代岩石圈地幔置换**：华北克拉通东部至少在<100 km深度内，古生代存在的古老、厚且难熔的太古宙克拉通型岩石圈地幔，在新生代被年轻、饱满、类似于大洋型的地幔所完全置换 [Zhu 2012, pp. 6-8, 9-10]。**证据**：新生代玄武岩中橄榄岩包体显示低Fo值（<91）和年轻Re-Os年龄，与古生代金伯利岩中的高Fo值（>92）太古宙地幔包体形成鲜明对比 [Zhu 2012, pp. 6-8]。
3.  **不同构造位置的地壳响应差异**：在克拉通破坏背景下，不同区域的下地壳经历了不同方式的改造。[[太行山]]地区以底侵作用为主，形成了弥散的壳幔过渡带；[[燕山]]地区则可能发生了下地壳拆沉，形成了尖锐的壳幔边界 [Zhu 2012, pp. 4-6]。**证据**：接收函数波形显示太行山PpPs震相弥散，而燕山PpPs震相尖锐 [Zhu 2012, pp. 4-6]。**限制**：这种差异化的动力学过程需要更多岩石学和地球化学证据的验证。
4.  **太平洋板块俯冲的主导动力学角色**：太平洋俯冲被认为是触发和控制华北克拉通破坏的主要动力学因素 [Zhu 2012, pp. 1-2]。**证据**：未从索引片段中确认具体证据链，仅引述为研究项目的主要结论之一。

## Open Questions
- 克拉通破坏背景下，不同机制（如拆沉、侵蚀、水弱化）在何时、何地起主导作用，其相互作用如何，需要进一步研究 [Zhu 2012, pp. 4-6]。
- 华北克拉通东部晚中生代-新生代岩石圈地幔从富水到贫水的转变过程和原因，需要补充说明 [Zhu 2012, pp. 1-2]。
- 苏鲁造山带下地壳麻粒岩包体的古元古代-太古宙锆石年龄，究竟代表古老下地壳的存留还是来自碰撞带的碎屑锆石，其蕴含的构造意义尚不明确 [Zhu 2012, pp. 8-9]。
- 从索引片段中确认，该论文并未得出古太平洋板块俯冲历史的具体细节，该方面仍待进一步研究 [Zhu 2012, pp. 6-6]。
