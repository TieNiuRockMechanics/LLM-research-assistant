# 二维孔隙裂隙双重介质逾渗规律研究.

## One-line Summary
将裂隙作为渗透通道引入逾渗理论，通过二维数值试验建立了孔隙裂隙双重介质逾渗阈值与孔隙率、裂隙分形维数、裂隙数量分布初值的定量关系。

## Research Question
如何将裂隙这一主要渗流通道纳入逾渗理论框架，描述孔隙‑裂隙双重介质的逾渗规律及其临界条件？[Feng 2007, pp. 1-5]

## Study Area / Data
数值试验在 `L × L`（取 `L = 250`）的二维正方形网格上进行，网格为孔隙‑裂隙双重介质模型。孔隙随机分布；裂隙按二维裂隙迹线分形规律生成，服从 `N(δ) = N₀·δ⁻ᴰ` [Feng 2007, pp. 1-5]。孔隙率 `n` 从 0 到 1 变化，分形维数 `D` 和裂隙数量分布初值 `N₀` 取多个水平进行组合计算。

## Methods
1. **模型构建**：在正方形网格上，孔隙按给定孔隙率随机占据格点（空格=1，实体=0）；裂隙按 `N(δ) = N₀·δ⁻ᴰ` 生成各级尺度裂隙并随机分布，落入格子的裂隙长度大于格子尺度1/2时记为空格。孔隙与裂隙的 `[0,1]` 分布按 `0+0=0, 0+1=1, 1+1=1` 叠加，形成双重介质模型 [Feng 2007, pp. 1-5]。
2. **逾渗概率定义**：最大连通团的空格数 `M(L)` 除以总格点数 `L²` 作为逾渗概率 `P(n, N₀, D) = M(L)/L²` [Feng 2007, pp. 1-5]。
3. **阈值识别**：通过改变 `n`、`D`、`N₀`，寻找最大连通团跨越边界时对应的临界参数组合，并对临界值进行回归分析，获得定性规律和定量表达式 [Feng 2007, pp. 1-5] [Feng 2007, pp. 5-6]。

## Key Findings
1. **孔隙介质**：无裂隙（`N₀=0`）时，逾渗临界孔隙率约为 `59.3%` [Feng 2007, pp. 1-5]。
2. **裂隙介质**：`n=0, N₀=e`（e≈2.71828）时，逾渗转变的临界分形维数介于 `1.6` 到 `1.68` 之间 [Feng 2007, pp. 1-5]。
3. **双重介质**：
   - 孔隙率 `n=30%, N₀=e` 时，临界分形维数介于 `1.5` 到 `1.6` 之间 [Feng 2007, pp. 1-5]。
   - 固定 `n=0`，`N₀` 与临界分形维数 `Dc` 的关系为 `N₀ = 8194.4 exp(-4.6493 Dc)`，相关系数 `r²=0.995` [Feng 2007, pp. 1-5]。
   - 固定 `D=1`，临界孔隙率 `nc` 与 `N₀` 的关系为 `nc = 0.59275 - 0.0097 N₀`，相关系数 `r²=0.9976` [Feng 2007, pp. 1-5]。
   - 固定 `N₀=e`，临界孔隙率 `nc` 与 `D` 的关系为 `nc = 0.59725 - 0.0002 exp(4.6578 D)`，相关系数 `r²=0.9916` [Feng 2007, pp. 1-5]。
4. **逾渗阈值表达式**：通过引入等效孔隙率函数，得到统一的逾渗临界条件
   `f(n, N₀, D) = nc_p - n - N₀ exp(-9.8687) exp(4.7917 D)`，
   其中 `nc_p` 为纯孔隙介质临界孔隙率。当 `f < 0` 时介质不发生逾渗，`f ≥ 0` 时出现跨越团，介质可渗透 [Feng 2007, pp. 5-6]。
5. **边缘情形**：当仅有一条裂隙（`N₀=1, D=1`）时，临界孔隙率约为 `0.59`，与纯孔隙介质几乎相同，裂隙对渗流概率无根本性影响 [Feng 2007, pp. 5-6]。

## Limitations
- 仅研究二维平面正方形网格模型，未涉及其他网格形状（三角形、六边形等）及三维情况 [Feng 2007, pp. 1-5]。
- 裂隙采用分形迹线描述，位置和方位随机分布，未考虑力学成因、裂隙开度及连通性的复杂特征。
- 数值试验网格尺寸固定为 250×250，未讨论有限尺度效应对临界值的影响程度。
- 回归公式基于特定参数区间的数值试验拟合，外推需谨慎。

## Assumptions / Conditions
- 模型为二维正方形网格，网格尺寸 `L=250` [Feng 2007, pp. 1-5]。
- 孔隙随机分布；裂隙数量服从分形关系 `N(δ) = N₀·δ⁻ᴰ`，裂隙落入格子且长度大于半格时视为孔隙 [Feng 2007, pp. 1-5]。
- 空格与实体按 “0+0=0, 0+1=1, 1+1=1” 叠加 [Feng 2007, pp. 1-5]。
- 逾渗概率定义为最大连通团占比；无边界效应说明。
- 回归分析中相关系数较高，但仅反映所用数据范围内的相关性。

## Key Variables / Parameters
- **孔隙率** `n`：介质中孔隙所占比例 (0~1)
- **裂隙分形维数** `D`：描述裂隙数量与尺度关系的分形参数
- **裂隙数量分布初值** `N₀`：分形关系中的系数
- **最大连通团大小** `M(L)`
- **等效孔隙率** `ne`
- **纯孔隙临界孔隙率** `nc_p` (取 0.59725 或 0.59275)

## Reusable Claims
1. **Claim 1**: 在无裂隙的二维正方形网格孔隙介质中，逾渗临界孔隙率约为 59.3%，与经典值 59.275% 一致。条件：二维正方形网格，域尺寸足够大 [Feng 2007, pp. 1-5]。
2. **Claim 2**: 裂隙的存在降低逾渗转变所需的孔隙率；反之，孔隙率增加会降低发生逾渗的裂隙临界分形维数。条件：双重介质，裂隙按分形随机分布 [Feng 2007, pp. 1-5]。
3. **Claim 3**: 孔隙裂隙双重介质的逾渗阈值可由 `f(n, N₀, D) = nc_p - n - N₀ exp(-9.8687) exp(4.7917 D)` 描述，`f≥0` 表示可渗透。适用条件：基于二维正方形网格数值拟合，`nc_p` 取 0.59725 [Feng 2007, pp. 5-6]。
4. **Claim 4**: 当 `D=1` 且 `N₀ ≥ e⁴` 时，逾渗概率随孔隙率近似线性变化，不再存在明显的临界点 [Feng 2007, pp. 1-5]。
5. **Claim 5**: 单一裂隙（`N₀=1, D=1`）对介质的逾渗概率没有根本性影响，临界孔隙率与纯孔隙介质相当（约 0.59）。条件：二维正方形网格 [Feng 2007, pp. 5-6]。
