# 基于正交各向异性介质理论的地应力地震预测方法.

## One-line Summary
基于正交各向异性介质理论，推导了一种新的水平应力差异比（ODHSR）地震预测方法，同时考虑了页岩地层的VTI和HTI各向异性特征，以提高页岩气压裂可压裂性评估的准确性。

## Research Question
如何在地应力地震预测中同时考虑页岩储层的VTI（水平层理）和HTI（垂直裂缝）各向异性特征，以解决现有HTI介质预测模型（DHSR）欠缺对水平层理特征考虑的问题，并消除累积误差。

## Study Area / Data
- **工区**：中²国东部某页岩储层 [Ma 2017, pp. 6-9]
- **数据源**：A井的纵波速度（V_P0）、横波速度（V_S0）、密度（ρ）以及各向异性参数 ε(V)、δ(V)、γ(V)和γ [Ma 2017, pp. 6-9]
- **参数获取方法**：HTI介质各向异性参数 ε(V)、δ(V)、γ(V)由构建的裂缝岩石物理等效模型估测得到；VTI介质各向异性参数γ由HTI介质的各向异性参数γ(V)经过转化得到 [Ma 2017, pp. 6-9]

## Methods
1.  **理论推导**：
    *   基于胡克定律的一般形式，推导正交各向异性（Orthorhombic Anisotropy, OA）介质的应力-应变本构关系 [Ma 2017, pp. 1-2, 3-5]。
    *   在假设地下岩石有界且水平方向应变为零（ε_x = ε_y = 0）的条件下，利用OA介质的柔度矩阵推导出最大水平应力、最小水平应力和水平应力差异比（ODHSR）与弹性参数及各向异性参数的关系 [Ma 2017, pp. 3-5]。
    *   定义了正交各向异性水平应力差异比（ODHSR），其为最大和最小水平应力变化差与最大水平应力的比值 [Ma 2017, pp. 1-2, 3-5]。
    *   利用Schoenberg和Sayers的线性滑动理论，将OA介质柔度矩阵近似为VTI背景介质柔度矩阵与扰动裂缝柔度矩阵之和，推导出OA介质主应力及ODHSR的近似公式 [Ma 2017, pp. 3-5]。
2.  **公式验证**：
    *   将ODHSR精确公式退化为HTI介质的DHSR公式，并与直接从HTI介质本构方程推导的DHSR进行比较 [Ma 2017, pp. 5-6]。
    *   将ODHSR近似公式退化为HTI介质的DHSR式，并与Gray（2011）提出的DHSR公式进行比较 [Ma 2017, pp. 5-6]。
3.  **实例验算**：利用中国东部某页岩储层A井的测井数据，计算并对比Gray（2011）的DHSR、本文精确公式退化的DHSR及本文的ODHSR近似公式结果 [Ma 2017, pp. 6-9]。

## Key Findings
1.  **公式一致性**：本文推导的ODHSR精确公式退化为HTI介质DHSR时，与通过HTI介质本构方程直接推导的DHSR计算公式完全相同 [Ma 2017, pp. 5-6]。
2.  **消除累积误差**：本文方法直接利用弹性参数和各向异性参数计算ODHSR，省略了先计算岩石力学参数再计算应力的步骤，减小了累积误差 [Ma 2017, pp. 6-9]。
3.  **实例曲线趋势**：精确公式退化的DHSR曲线与Gray的DHSR曲线趋势基本一致，证明了ODHSR精确公式的合理性；ODHSR近似公式与Gray的DHSR曲线趋势基本一致，但ODHSR高值部分略高于DHSR [Ma 2017, pp. 6-9]。
4.  **VTI特征影响**：考虑VTI各向异性参数γ的影响会导致ODHSR与DHSR在高值部分产生差异；当γ置零（消除VTI特征）时，ODHSR与DHSR结果一致 [Ma 2017, pp. 6-9]。
5.  **压裂潜力评估**：ODHSR值越低，表明该位置易压裂成网；高值则表示易形成定向排列的裂缝 [Ma 2017, pp. 6-9]。

## Limitations
*   **构造复杂度**：ODHSR公式是在水平两个方向应变为零的假设条件下推导得到的，因此该公式适用于构造不复杂的页岩储层 [Ma 2017, pp. 3-5, 6-9]。
*   **公式复杂性**：本文推导的ODHSR精确公式过于复杂，不便于实际应用 [Ma 2017, pp. 6-9]。
*   **未考虑构造应力**：研究未考虑构造应力对地应力的影响 [Ma 2017, pp. 6-9]。

## Assumptions / Conditions
*   假设地下岩石是有界的、不能移动的，因此水平方向上的应变为零（ε_x = ε_y = 0） [Ma 2017, pp. 3-5]。
*   存在垂直方向的主应力和两个水平方向的应力 [Ma 2017, pp. 3-5]。
*   正交各向异性（OA）介质模型由水平层状互层矿物（VTI特征）和沿水平对称轴方向排列的裂缝（HTI特征）共同构成 [Ma 2017, pp. 3-5]。
*   线性滑动理论可用于将OA介质柔度矩阵近似为VTI背景柔度矩阵与裂缝扰动柔度矩阵之和 [Ma 2017, pp. 3-5]。

## Key Variables / Parameters
*   **σ_x (σ_h)**：最小水平地应力 [Ma 2017, pp. 3-5]
*   **σ_y (σ_H)**：最大水平地应力 [Ma 2017, pp. 3-5]
*   **σ_z (σ_v)**：垂直地应力 [Ma 2017, pp. 3-5]
*   **ODHSR**：正交各向异性水平应力差异比（Orthorhombic Differential Horizontal Stress Ratio），定义式为 (σ_y - σ_x) / σ_y [Ma 2017, pp. 1-2, 3-5]
*   **V_P0**：准纵波垂向速度 [Ma 2017, pp. 6-9]
*   **V_S0**：极化方向为x方向的准横波垂向速度 [Ma 2017, pp. 6-9]
*   **ρ**：密度 [Ma 2017, pp. 3-5]
*   **ε, δ, γ**：VTI介质的Thomsen各向异性参数 [Ma 2017, pp. 3-5]
*   **ε(V), δ(V), γ(V)**：HTI介质的各向异性参数 [Ma 2017, pp. 3-5]
*   **Z_N**：裂缝的法向柔度 [Ma 2017, pp. 3-5]
*   **Δ_N**：裂缝的法向弱度 [Ma 2017, pp. 3-5]

## Reusable Claims
*   **Claim 1**: 基于正交各向异性（OA）介质的本构方程和水平应变为零的假设，可以推导出同时包含VTI和HTI特征的**水平应力差异比（ODHSR）**，以评估页岩储层的可压裂性。[Ma 2017, pp. 1-2, 3-5]
    *   **Condition**: 适用于构造不复杂的页岩储层。
    *   **Limitation**: 未考虑构造应力，水平应变为零的假设限制了其应用范围。
*   **Claim 2**: 本文推导的**正交各向异性水平应力差异比（ODHSR）**在退化为HTI介质时，与从HTI本构方程直接推导的**DHSR精确公式**一致，消除了Gray（2011）基于线性滑动理论简化带来的误差，并且直接从弹性参数和各向异性参数计算，减小了累积误差。[Ma 2017, pp. 5-6, 6-9]
    *   **Condition**: 需要HTI介质的各向异性参数和弹性参数作为输入。
*   **Claim 3**: ODHSR近似公式中**VTI介质各向异性参数**（如γ）的引入，会导致ODHSR预测值在局部区域高于仅考虑HTI特征的DHSR，说明考虑水平层理特征对地应力预测有重要影响。[Ma 2017, pp. 6-9]
    *   **Condition**: 实际数据计算中，VTI和HTI参数需从井资料或地震反演中获得。

## Connections To Other Work
*   继承并改进了Gray等人（2010a, 2010b, 2010c, 2011, 2012）基于**HTI介质**利用地震数据进行地应力预测和提出DHSR的方法，本文认为该方法“只基于HTI介质，没有考虑介质的水平层理影响”，且通过弹性参数计算岩石力学参数再求DHSR“增加了累积误差” [Ma 2017, pp. 2-3]。
*   沿用了Thomsen（1986）提出的**VTI介质各向异性参数**（ε, δ, γ）和Rüger（1996）提出的**HTI介质各向异性参数**（ε(V), δ(V), γ(V)) [Ma 2017, pp. 3-5]。
*   引用了Tsvankin（1997）关于**正交各向异性介质**弹性性质的研究，并利用其定义的各向异性参数表征方式来推导ODHSR公式 [Ma 2017, pp. 3-5, 6-9]。
*   应用了Schoenberg和Sayers（1995）提出的**线性滑动理论**，将正交各向异性介质的柔度矩阵近似为VTI背景与裂缝柔度之和 [Ma 2017, pp. 3-5]。
*   使用了Iverson（1995）提出的假设：存在垂直方向的主应力和两个水平方向的应力，且地下岩石有界，水平应变为零 [Ma 2017, pp. 3-5]。
*   与Gray（2011）提出的**HTI介质主应力及DHSR公式**进行了对比验证，证明了本文推导的合理性 [Ma 2017, pp. 6-9]。
