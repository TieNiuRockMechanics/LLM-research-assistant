# Fracture Size Estimation Using Data from Multiple Boreholes.

## One-line Summary
基于椭圆裂隙假设，推导了利用多钻孔交切计数估计裂隙特征尺寸（半长轴）均值、标准差及推断其总体分布（分形、指数或对数正态）的解析公式与方法，并通过蒙特卡罗模拟进行了验证。

## Research Question
如何从多个钻孔壁上采集到的裂隙与钻孔的交切数目数据，推断三维岩体中椭圆形裂隙的尺寸（半长轴）的概率密度函数（PDF）？—— 核心挑战在于单钻孔壁上的迹线信息（尤其是当钻孔直径远小于裂隙尺寸时）难以直接转换为尺寸分布；本文探索利用相邻平行钻孔中交切裂隙数量的差异来间接恢复尺寸分布。

## Study Area / Data
本研究未使用特定野外工点数据，所有验证均通过自主开发的蒙特卡罗模拟程序RJNS3D（Rock Mass Joint Network Simulation in Three Dimensions）完成。模拟中裂隙中心服从泊松分布，取向采用Fisher分布或确定性取向，尺寸分布分别选用指数、均匀、正态、对数正态、分形等类型，裂隙密度p₃₀、平均尺寸μₐ、钻孔倾角β等参数均在多组敏感性分析中变化（见表1、表2）。模拟域大小为40×40×40或80×80×80，钻孔半径c=1，深度H=40或80，钻孔间距D=10，平行钻孔5个，按投影椭圆长轴方向等距布置。[Gao 2016, pp. 8-10]

## Methods
### 1. 基础假设与投影
- 所有裂隙视为无厚度的椭圆平面，同一组内裂隙的纵横比κ（长轴/短轴）相同，但不要求裂隙彼此平行（在推断分布时额外假定裂隙平行）。
- 裂隙中心位置服从泊松分布（密度p₃₀），取向由倾向α、倾角β和旋转角γ定义，服从二维PDF（如Fisher分布），尺寸与位置、取向独立。
- 钻孔直径远小于裂隙尺寸。
- 将每个椭圆裂隙沿钻孔轴线方向投影到垂直于钻孔轴线的平面上，结果仍为椭圆（式(1)~(14)），其半长轴a′和半短轴b′由κ、钻孔方向与裂隙法向关系决定，并获得投影椭圆旋转角ω。[Gao 2016, pp. 2-5]

### 2. 均值与标准差的单孔估计
基于Mauldon & Mauldon (1997)的包络面积推导，给出了全交切和部分交切的面积函数S(a, α, β)与Sf(a, α, β)（式(15)、(16)）。对于给定尺寸a的裂隙，其与钻孔交切的期望数量为：
\[
N_I = p_{30} H \int_0^{2\pi}\int_0^{\pi/2} \int_{\xi_c/\xi_a}^{\infty} S(a,\alpha,\beta)f(\alpha,\beta)\, g(a)\, da\, d\beta\, d\alpha
\]
利用全交切与部分交切的差值消去含c²项，在d≪a的近似下导出特征尺寸（半长轴a）的均值μₐ和标准差σₐ的估算式（式(23)和(26)）。这些式子因始终在投影平面上推导，自动修正了取向偏差，无需额外校正因子。[Gao 2016, pp. 5-5]

### 3. 多孔平行钻孔推断尺寸分布
假设多个等间距D的平行钻孔沿投影椭圆长轴方向排列，且裂隙均相互平行。考虑裂隙尺寸落入不同区间时，其中心落在不同重叠区域将导致交切1个、2个、…、nb个钻孔。推导出裂隙与ζ个相邻钻孔交切的期望数量依赖于尺寸a的函数Sgray(a)（式(33)~(37)）。
给定实测的交切计数^N₁, ^N₂, …, ^Nₙb，先由式(23)、(26)得到各孔μₐ,i、σₐ,i再取平均(式(38))，然后分别代入分形（幂律）、指数或对数正态分布形式(式(39))，计算出理论期望值〈Nᵢ〉，与实测^Nᵢ比较，最匹配的分布即为推断结果。[Gao 2016, pp. 6-9]

### 4. 蒙特卡罗验证
利用RJNS3D工具箱生成不同输入（尺寸分布类型、裂隙密度p₃₀、取向β、平均尺寸μₐ）的确定性取向裂隙网络，对单孔均值估计进行20次重复模拟，验证式(23)的精度与敏感性；对多孔分布推断则通过比较^N与〈N〉验证所选分布的正确性。参数设置详见表1和表2。[Gao 2016, pp. 8-13]

## Key Findings
- **单孔均值估计**：式(23)在裂隙密度p₃₀≥0.01、平均尺寸满足μₐ≥cξₐ/ξb²（曲率限制）且钻孔不平行于裂隙面的条件下具有足够精度，对不同尺寸分布（指数、均匀、正态、对数正态）均给出接近真实的估计。估计值对指数分布略有低估（因指数分布样本均值可能偏小）。当p₃₀=0.001时误差显著增大（LS residual=18.29）。[Gao 2016, pp. 8-9]
- **多孔分布推断**：针对指数、分形、对数正态三种输入，估计的〈Nᵢ〉与模拟采样值^Nᵢ匹配良好，方法对分布类型不敏感；密度p₃₀≥0.01时能正确识别分布，当p₃₀=0.001时无法得出结论；对不同的倾角（20°,40°,60°）和平均尺寸（5,10,15）同样表现出稳健性，惟当μₐ估计偏大时会导致N曲线整体上移。[Gao 2016, pp. 10-13]

## Limitations
- 推断尺寸分布时要求所有裂隙相互平行，与一般工程中同一组内仅统计平行的假设一致，但未考虑组内产状离散。
- 仅适用于三种预设分布（分形、指数、对数正态），若真实分布不属于其中之一，可能给出次优匹配。
- 多孔方法要求钻孔沿投影椭圆长轴方向等距布置，实际钻孔布设可能无法严格满足。
- 低裂隙密度（p₃₀<0.01）下均值估计和分布推断均不可靠。
- 假设钻孔直径远小于裂隙尺寸，若存在大量小裂隙则不成立。
- 文中方法仅针对铅直钻孔验证，对任意方向钻孔的通用性未检验，但推导本身允许通用钻孔方向（通过单位向量b定义）。
- 未考虑裂隙厚度、填充物、测量误差等因素。

## Assumptions / Conditions
- 裂隙呈无厚度椭圆平面，同一组内κ恒定，旋转角γ为定值。
- 裂隙中心在三维空间服从泊松分布（均匀随机）。
- 裂隙尺寸a的概率密度g(a)与位置、取向独立。
- 裂隙取向可用二维PDF（如Fisher分布，式(18)）描述。
- 钻孔半径c远小于裂隙尺寸；全交切要求裂隙最大曲率处小于钻孔曲率，即a ≥ cξₐ/ξb²。
- 在推断尺寸分布时补充假定：所有用于推断的裂隙严格平行，且裂隙平行于投影椭圆长轴，钻孔沿长轴等间距排列（使重叠面积计算可行）。
- 投影椭圆用椭圆E近似曲线L（式(29)），当a′→b′时误差可忽略，在工程应用中认为有效。
- 推导的期望值公式基于无限域统计，实际样本存在随机波动。

## Reusable Claims
1. **单孔均值估计公式**(式23)：通过统计同一钻孔壁上全交切和部分交切的裂隙数之差，可在已知裂隙密度p₃₀、钻孔长度H、裂隙取向分布f(α,β)及投影几何参数（由κ与钻孔方向决定）的条件下，估算椭圆裂隙半长轴的均值μₐ。
   - *条件*：p₃₀ ≥ 0.01；裂隙不平行钻孔；c ≪ μₐ且μₐ ≥ cξₐ/ξb²；旋转角γ、κ已知或可通过分析确定。
   - *限制*：对指数分布可能低估，因为随机样本均值天然小于理论均值。
2. **单孔标准差估计公式**(式26)：基于全交切数〈NⅠ〉、均值μₐ和钻孔几何参数可近似求得σₐ，用于后续分布推断的辅助信息。
   - *条件*：同均值估计，且需要准确计数全交切数目。
3. **多孔平行钻孔分布推断策略**：在钻孔沿投影椭圆长轴等距排列、裂隙均平行且符合分形、指数或对数正态之一的条件下，通过比较实测的与不同相邻钻孔交切的数量向量(^N₁,…,^Nₙb)与三种候选分布的理论期望值，可选定最可能的尺寸分布g(a)。
   - *条件*：n_b ≥ 3；p₃₀ ≥ 0.01；钻孔严格平行且等距；裂隙取向完全一致（平行）；投影椭圆长轴与钻孔连线重合。
   - *限制*：若真实分布不在预设三类中，结果仅为最佳拟合；低密度下无解。
4. **投影面积修正**：全交切和部分交切的有效中心面积S、Sf及其差值（式(22)）自动消除了钻孔半径c的干扰，使得在c≪a假设下能简便获得μₐ的解析式，且无需额外取向偏差校正。该投影思路可推广至其他形状裂隙的钻孔交切问题。
5. **迹线分类**：钻孔壁上裂隙迹线分为全正弦曲线（全交切）和不完整正弦曲线（部分交切），这一区分是推导中划分Sf与S的依据，可应用于实际钻孔图像解译中的裂隙分类统计。
6. **蒙特卡罗验证条件**：当取p₃₀=0.05、κ=2、γ=0、垂直钻孔、β=15°~30°，且a的变异系数适中时，公式(23)给出的μₐ估计误差小于15%（见表1多种分布下LS residual均较小）。可作为类似工程中方法适用性的先验参考。

## Open Questions
- 本文方法局限于平行钻孔且严格沿投影长轴排列的情形，如何将这些解析关系拓展到任意方向的随机钻孔阵列，文中第5节指出将在后续论文中阐述。[Gao 2016, pp. 13]
- 实际裂隙组内产状存在离散，Fisher分布参数τ的作用仅在均值公式中作为积分权重出现，多孔推断时强制平行可能引入多大误差，尚需定量评估。
- 当裂隙尺寸分布不是分形、指数或对数正态时，能否利用更多矩（如偏度）构造非参数推断方法？
- 低密度裂隙网络（p₃₀<0.01）是否有其他统计量（例如交切间距）可以辅助推断尺寸？
- 钻孔直径不可能无限小，c/a′比值大到何种程度会导致投影近似及椭圆弧长近似(式(29))失效，未给出临界条件。
- 文中所有验证限于铅垂钻孔，实际工程中倾斜钻孔交切计数能否直接套用公式，需确认投影系数kij的计算不受奇异影响。
