# Destruction of the North China Craton.

## One-line Summary
华北克拉通破坏是一次以岩石圈减薄为基础、伴随岩石圈性质与热状态转变的大规模地质事件，其核心驱动力为太平洋板块西向俯冲引起的构造‑岩浆活动，破坏峰期出现在早白垩世，并表现出显著的空间非均一性（东部为主）。

## Research Question
华北克拉通（NCC）破坏的时空分布、深浅结构响应、动力学机制及其在全球大陆演化中的意义。核心科学问题包括：破坏的时空范围、深部结构与热‑流体过程、浅表地质‑成矿‑地震响应、破坏机制与动力学、以及克拉通破坏在板块构造中的角色。

## Study Area / Data
研究区为整个华北克拉通，重点在东部块体、中部带（太行山重力梯度带附近）及周边造山带。主要数据来源：
- 975 台流动宽频带地震仪组成的密集台阵（平均台间距 10–15 km），三条总长 3400 km 的宽角反射/折射剖面，以及渤海湾区域海底地震仪‑陆地联合探测剖面（总长约 930 km）[Zhu 2012, pp. 2-4]。
- 不同时代（古生代金伯利岩、中生代火山岩、新生代玄武岩）携带的幔源包体与捕掳晶，以及中生代‑新生代火成岩样品[Zhu 2012, pp. 6-8, 8-9, 9-10]。
- 已有的中国国家数字地震台网及区域固定台站数据[Zhu 2012, pp. 2-4, 4-6]。

## Methods
采用多学科综合观测与分析方法：
- **地震学**：接收函数偏移成像（P‑s 和 S‑p 转换波）刻画岩石圈‑软流圈边界及壳内结构[Zhu 2012, pp. 2-4, 4-6]；体波层析成像（P 波、S 波、Vp/Vs）获取上地幔三维速度结构[Zhu 2012, pp. 6-6]；瑞利面波层析成像约束 S 波速度[Zhu 2012, pp. 6-6]；剪切波分裂测量得到上地幔各向异性[Zhu 2012, pp. 6-6]；对 410 km 和 660 km 间断面成像[Zhu 2012, pp. 6-6]。
- **岩石学与地球化学**：利用幔源包体/捕掳晶的常量元素（如橄榄石 Fo 值）、放射性成因同位素（Re‑Os、Sr‑Nd‑Pb‑Hf）和非传统稳定同位素（Li、Mg、Fe）示踪岩石圈地幔性质与改造过程[Zhu 2012, pp. 6-8, 8-9, 9-10]；通过傅里叶变换红外光谱（FTIR）和离子探针（SIMS）测量名义无水矿物中的水含量[Zhu 2012, pp. 10-11, 17-17]。
- **构造地质学**：盆地分析、变质核杂岩及断层运动学与岩脉分布重建伸展方向演变[Zhu 2012, pp. 11-13, 13-13, 14-15]。

## Key Findings
1. **破坏的空间非均一性**：岩石圈厚度自东南向西北加深，从东南部的 60–70 km 增至西北部的 90–100 km，鄂尔多斯盆地仍保留约 200 km 厚岩石圈；地壳与岩石圈结构显示东部强烈改造，而西部相对稳定，破坏范围限于东部太行山重力梯度带以东[Zhu 2012, pp. 2-4]。
2. **岩石圈地幔的时代差异与水含量变化**：古生代地幔具典型克拉通特征（高 Fo、太古宙 Re‑Os 年龄）；中生代地幔转变成以二辉橄榄岩为主，富水（>1000 ppm）；新生代地幔极度贫水（<50 ppm）且经历强烈的熔‑岩反应（再肥化）[Zhu 2012, pp. 6-8, 8-9, 9-10, 10-11]。
3. **浅表‑深部过程的联动**：早白垩世（~120–130 Ma）的伸展构造（变质核杂岩、裂陷盆地）与大规模岩浆活动峰值在时间上一致，证实浅表变形受克拉通破坏过程控制；此后构造‑岩浆活动减弱，晚白垩世以区域性隆升为主[Zhu 2012, pp. 11-13, 13-13]。
4. **太平洋俯冲主导的动力学机制**：主要证据包括（a）东部 NNE 向伸展方向与太平洋板块运动方向同步顺时针旋转；（b）100 Ma 以来的基性岩浆中出现具 HIMU 特征的洋壳组分；（c）晚中生代岩石圈地幔显著富水，指示俯冲板片脱水注入[Zhu 2012, pp. 14-15, 15-16, 16-17, 17-17]。

## Limitations
- 岩石样品的空间分布不连续，限制了对岩石圈破坏范围与性质的全面约束 [Zhu 2012, pp. 2-4]。
- 关于华南陆壳物质是否参与岩石圈改造仍有争议：部分学者认为来自扬子俯冲地壳，另一部分认为源自华北下地壳拆沉 [Zhu 2012, pp. 8-9]。
- 太平洋俯冲影响在时空上的精准起始时间尚不明确；目前仅从岩浆记录推断其影响可能早于晚白垩世 [Zhu 2012, pp. 16-17]。
- 本文为综述性进展汇报，未包含新模型的正演或反演定量模拟，诸多动力学过程仍定性描述。

## Assumptions / Conditions
- 克拉通破坏的定义是整体丧失克拉通属性（不仅仅是岩石圈减薄） [Zhu 2012, pp. 2-2]。
- 岩石圈减薄发生于区域性伸展背景下，浅表广泛而强烈的伸展是克拉通破坏的关键浅表标志 [Zhu 2012, pp. 11-13]。
- 板片俯冲脱水增加上覆岩石圈的含水量，从而降低岩石圈粘度，这是水弱化模型的基本假设 [Zhu 2012, pp. 10-11]。
- 华北克拉通在 1.8 Ga 拼合后，从中元古代至早三叠世长期保持克拉通属性 [Zhu 2012, pp. 11-13]。

## Key Variables / Parameters
- **岩石圈厚度**：东南 60–100 km，西北 >100 km，鄂尔多斯 ~200 km；作为破坏范围的空间约束 [Zhu 2012, pp. 2-4]。
- **水含量**：中生代 >500 至 >1000 ppm；新生代 <50 ppm；控制岩石圈流变强度 [Zhu 2012, pp. 10-11, 17-17]。
- **岩浆活动峰期**：~125–120 Ma （破坏峰期） [Zhu 2012, pp. 11-13]。
- **伸展方向旋转**：早白垩世中期 NWW‑SEE → 晚白垩世‑古近纪 NS，与太平洋板块运动同步 [Zhu 2012, pp. 14-15]。
- **大洋板片组分识别**：δ¹⁸O 低值、HIMU‑OIB 微量元素特征、高 ²⁰⁶Pb/²⁰⁴Pb (>19.5) [Zhu 2012, pp. 15-16]。
- **岩石圈地幔演化**：古生代为高 Fo、太古宙 Re‑Os 年龄的难熔方辉橄榄岩 → 中生代为相对饱满的二辉橄榄岩/辉石岩 → 新生代为经历碳酸盐熔体再肥化的低 Fo 二辉橄榄岩/异剥橄榄岩 [Zhu 2012, pp. 6-8, 8-9, 9-10]。

## Reusable Claims
- **岩石圈厚度空间非均一性**：华北东部岩石圈厚度自东南向西北递增，形成鲜明的东西差异，且陡变带与南北重力梯度带重合。该格局限定了克拉通破坏的主要范围在东部，可推广为太平洋俯冲影响范围的重要指示。**条件**：需有密集宽频带地震数据，使用接收函数偏移成像 [Zhu 2012, pp. 2-4]。
- **古生代克拉通岩石圈的证据**：奥陶纪金伯利岩中的橄榄岩包体具有高 Fo（>92）和太古宙 Re‑Os 模式年龄，证明破坏前东部曾存在厚约 200 km 的古老、难熔岩石圈根。**条件**：适用于未受后期强烈交代的捕捞体记录 [Zhu 2012, pp. 6-8]。
- **水含量控制克拉通稳定性**：早白垩世岩石圈地幔水含量 >1000 ppm，粘度接近软流圈，对应破坏峰期；新生代水含量 <50 ppm，克拉通复稳。因此，俯冲板片释放的水是克拉通去稳的关键因素。**条件**：需要排除岩浆上升过程中地壳混染，使用最早的结晶矿物（高 Mg# 单斜辉石） [Zhu 2012, pp. 10-11, 17-17]。
- **岩浆源区洋壳组分的识别**：100–40 Ma 基性岩表现出随时间递进的化学变化（Eu/Eu* 升高、⁸⁷Sr/⁸⁶Sr 降低）以及 Indian‑MORB 型 Pb 同位素，可作为识别俯冲洋壳对大陆内部岩浆贡献的判据。**条件**：适用于无显著地壳混染的基性熔岩 [Zhu 2012, pp. 16-17]。
- **伸展方向与板块运动的耦合**：华北东部伸展方向从早白垩世中期至新生代的顺时针旋转与太平洋板块运动方向同步变化，表明弧后伸展背景下克拉通破坏受控于板块边界动力学。**条件**：需综合盆地、变质核杂岩及断层运动学数据 [Zhu 2012, pp. 14-15]。

## Connections To Other Work
- 与全球其他克拉通（如南非克拉通、怀俄明克拉通）对比，揭示只有受大洋俯冲强烈影响的克拉通才会发生“破坏”，而非单纯减薄（如印度克拉通因地幔柱减薄但保持克拉通属性）[Zhu 2012, pp. 17-17]。
- 参照北美科迪勒拉地区，法拉隆板块东向俯冲导致北美克拉通部分破坏，类似的弧后区域构造背景 [Zhu 2012, pp. 17-17]。
- 与华南大陆的长期多期次“再活化”对比，华南代表另一种大陆演化途径，对理解大陆再造与破坏的差异性有重要意义 [Zhu 2012, pp. 19-19]。
- 早期提出的“燕山运动”、“地台活化”等概念为现今的克拉通破坏研究提供了基础框架 [Zhu 2012, pp. 1-2]。

## Open Questions
- 中生代‑新生代基性岩源区中的水和俯冲板片组分的精确来源（太平洋俯冲 vs. 其他俯冲系统）以及太平洋俯冲历史尚不清晰 [Zhu 2012, pp. 19-19]。
- 俯冲大洋板片如何与岩石圈地幔发生反应，导致其富水及后期脱水的物理化学过程不明 [Zhu 2012, pp. 19-19]。
- 岩石圈地幔向软流圈地幔转变（及反向过程）的精细机制和过渡带的性质需要更多约束 [Zhu 2012, pp. 19-19]。
- 华北克拉通南、北边缘碰撞（扬子与蒙古块体）对破坏的贡献及与太平洋俯冲的协同作用有待量化 [Zhu 2012, pp. 13-15]。
- 克拉通破坏与大陆再造（reworking）的根本区别与联系，以及它们在全球大陆演化中的普遍性仍待验证 [Zhu 2012, pp. 19-19]。
- 破坏结束后岩石圈的部分加厚过程（如古近纪裂陷）与岩石圈整体转变为厚度增生的关系需要高精度深部成像与岩石学联合研究 [Zhu 2012, pp. 13-13]。
