# 渤海变质岩潜山油藏储集层综合评价与裂缝表征.

## One-line Summary
本文综合利用岩心、薄片和成像测井等资料，对渤海JZ25-1S油田变质岩潜山油藏进行了储集层评价和裂缝表征研究，确定了储集层划分标准并评价了裂缝有效性。[Tong 2012, pp. 1-8]

## Research Question
如何对渤海海域变质岩潜山油藏进行有效的储集层综合评价与裂缝表征？[Tong 2012, pp. 1-8]

## Study Area / Data
研究区为渤海JZ25-1S油田的太古宇变质岩潜山油藏。[Tong 2012, pp. 1-8] 使用的数据包括岩心、薄片、成像测井（FMI）、原油包裹体检测、区域地应力方向及声波全波列测井资料。[Tong 2012, pp. 1-8]

## Methods
1.  **储集层划分**：将潜山储集层纵向上划分为风化壳、半风化壳和基岩3段，并划分为裂缝-孔隙型、孔隙-裂缝型和微裂缝-致密型3种储集层类型。[Tong 2012, pp. 1-8]
2.  **裂缝分类与表征**：根据倾角将裂缝分为低角度（<15°）、高角度（>75°）和倾斜裂缝（15°~75°）。[Tong 2012, pp. 1-8] 利用岩心观察和FMI图像识别裂缝类型及密度。[Tong 2012, pp. 1-8]
3.  **裂缝有效性评价**：
    *   利用原油包裹体检测技术，通过包裹体荧光特征判断裂缝充填与有效性。[Tong 2012, pp. 1-8]
    *   利用现今最大地应力方向，评价裂缝走向与应力方向的匹配关系。[Tong 2012, pp. 1-8]
    *   利用声波全波列测井资料，通过纵横波速度比等参数评价裂缝有效性。[Tong 2012, pp. 1-8]
4.  **储集层分类标准建立**：综合成像测井精细解释与生产实际，确定定性-半定量的储集层划分标准，将储集层划分为Ⅰ、Ⅱ、Ⅲ三个级别。[Tong 2012, pp. 1-8]

## Key Findings
1.  研究区半风化壳储集层最为发育，孔隙-裂缝型储集层为主要产层。[Tong 2012, pp. 1-8]
2.  裂缝的主要走向与潜山主断层走向大致平行。[Tong 2012, pp. 1-8]
3.  与现今最大地应力方向基本保持一致的晚期开启的倾斜裂缝最为有效。[Tong 2012, pp. 1-8]
4.  建立了将潜山储集层划分为Ⅰ、Ⅱ、Ⅲ三个级别的定性-半定量划分标准。[Tong 2012, pp. 1-8]

## Reusable Claims
1.  在变质岩潜山油藏中，半风化壳是储集层最发育的部位，孔隙-裂缝型储集层可作为主要产层。[Tong 2012, pp. 1-8] （条件：针对类似渤海JZ25-1S油田的变质岩潜山）
2.  裂缝的有效性受其形成时期和现今地应力场控制，晚期开启且走向与现今最大地应力方向一致的裂缝最为有效。[Tong 2012, pp. 1-8] （条件：适用于评价裂缝导流能力）
3.  综合成像测井解释与生产动态数据，可以建立定性-半定量的储集层级别划分标准。[Tong 2012, pp. 1-8] （条件：需要足够的测井和生产数据）
