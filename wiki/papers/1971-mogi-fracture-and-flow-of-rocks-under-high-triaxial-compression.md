# Fracture and Flow of Rocks under High Triaxial Compression.

## One-line Summary

本文开发了一种新的真三轴压缩实验技术，首次在三个主应力不等的一般三轴应力状态下系统研究了岩石的破裂与流动规律，发现破坏强度不仅取决于最小主应力 σ₃，还显著受中间主应力 σ₂ 影响，并提出了广义 von Mises 破裂准则和广义 von Mises 屈服准则，以及延性度对 (cσ₃ - σ₂) 的单调依赖关系，进而推测了上地幔的破裂与流动特性。

## Research Question

在真三轴应力 (σ₁ > σ₂ > σ₃) 条件下，岩石破裂与屈服的应力状态如何由三个主应力决定？延性度（断裂前永久应变）又是如何受 σ₂ 和 σ₃ 影响的？能否提出统一的破坏准则以推广经典 von Mises 理论，并据此推测上地幔的力学行为？

## Study Area / Data

- 实验材料：三种碳酸盐岩（Solenhofen 石灰岩、Dunham 白云岩、Yamaguchi 大理岩）和四种硅酸盐岩（Mizuho 粗面岩、Orikabe 二长岩、Inada 花岗岩、Manazuru 安山岩）。  
- 所有岩石“近乎均质各向同性，力学性质差异巨大” [Mogi 1971, pp. 6-7]。  
- 实验温度：室温（未明确控制温度）。  
- 加载速率：恒定轴向应变率 10⁻⁴/s [Mogi 1971, pp. 3-6]。

## Methods

### 实验装置：新真三轴压缩仪

- 采用“常规三轴实验的重要改进”：围压（最小主应力 σ₃）通过流体施加，轴向载荷（对应 σ₁）通过固体活塞施加，侧向载荷（对应 σ₂）也通过独立固体活塞施加 [Mogi 1971, pp. 2-3]。  
- 试样为矩形棱柱，尺寸 1.5 cm × 1.5 cm × 3.0 cm。  
- 轴向端块用环氧树脂连接，侧向端块通过润滑剂（聚四氟乙烯薄片或橡胶）与试样接触以减小摩擦，同时试样侧面包覆铜箔以防润滑剂侵入 [Mogi 1971, pp. 3-6]。  
- 整个试样用硅橡胶包裹，防止压力介质（油）渗入。  
- 围压最大可达 5 kb，轴向载荷通过 70 吨千斤顶，侧向载荷通过 30 吨千斤顶，载荷由内部测力传感器测量，精度约 2% [Mogi 1971, pp. 3-6]。  
- 围压测量使用锰铜计和 Heise Bourdon 管压力计，精度达数巴 [Mogi 1971, pp. 3-6]。  
- 实验步骤：先施加围压，再恒定侧向载荷（σ₂），最后以恒定应变率增加轴向载荷；未计加载顺序影响 [Mogi 1971, pp. 3-6]。

### 参数定义

- 主应力以压缩为正，σ₁ > σ₂ > σ₃。  
- 极限强度（破坏强度）取应力-应变曲线的最大差应力 (σ₁ - σ₃)。  
- 屈服强度取差应力-应变曲线拐点处的差应力，或产生一定永久应变的差应力。  
- 延性度 εᵖ 定义为断裂前永久轴向应变（总轴向应变减去弹性轴向应变）[Mogi 1971, pp. 6-7]。

## Key Findings

1. **破裂强度受 σ₂ 和 σ₃ 的双重影响**：在恒定 σ₃ 下，极限强度随 σ₂ 增大而显著增加；但延性度却随 σ₂ 增大而明显减小，而延性度随 σ₃ 增大而增大，二者影响方向相反 [Mogi 1971, pp. 6-7]。

2. **广义 von Mises 破裂准则**：  
   实验表明，不同 σ₃、σ₂ 组合的破裂点，若以八面体剪应力 τ_oct 对有效平均压力 (σ₁ + σ₃)/2 作图，各点汇聚为一条曲线，而与 σ₂ 值无关。破裂准则可写为：
   ```
   τ_oct = f₁((σ₁ + σ₃)/2)
   ```
   其中 f₁ 为单调递增函数 [Mogi 1971, pp. 7-10]。该准则可解释为：畸变应变能达到一临界值时发生破裂，该临界值随破裂面上的有效平均应力 (σ₁ + σ₃)/2 单调增加 [Mogi 1971, pp. 10-13]。

3. **广义 von Mises 屈服准则**：  
   在碳酸盐岩和粗面岩中观察到明显屈服，其屈服点满足：
   ```
   τ_oct = f₂((σ₁ + σ₂ + σ₃)/3)
   ```
   f₂ 亦为单调递增函数。当 f₂ 为常数时（如 Solenhofen 石灰岩）该准则退化为经典 von Mises 准则 [Mogi 1971, pp. 10-13]。物理意义：屈服发生时的临界畸变应变能随平均应力 (σ₁+σ₂+σ₃)/3 增加而增加 [Mogi 1971, pp. 10-13]。

4. **延性度经验关系**：  
   断裂前永久应变 εᵖ 可以表达为：
   ```
   εᵖ = f₃(cσ₃ - σ₂)
   ```
   其中 c 为岩石常数，表征 σ₃ 效应与 σ₂ 效应的相对权重。例如，Dunham 白云岩中 c ≈ 3，即 σ₂ 作用约为 σ₃ 的 1/3；Solenhofen 石灰岩中 c ≈ 20，σ₂ 作用极弱 [Mogi 1971, pp. 13-15]。

5. **对上地幔的推论**：  
   - 地幔域内岩石的屈服或破裂将在畸变应变能达到临界值时发生，该临界值随滑移面上的有效平均压力增大而升高 [Mogi 1971, pp. 15-15]。  
   - 深部地区较高的 σ₂ 会降低延性度，因此与常规三轴实验预期相比，发生脆性破裂（弹性应变能突然释放，即地震）的可能性更高 [Mogi 1971, pp. 15-15]。但该推论受限于实验压力和温度远低于地幔条件。

## Limitations

- 实验在室温和单一应变速率下进行，未考虑温度效应，温度是影响岩石脆-延转换的关键参数。  
- 最大围压 5 kb，远低于地幔实际压力（约 10~100 kb）。  
- 部分硅酸盐岩数据存在离散，准则的普适性有待更多岩石类型验证。  
- 屈服应力的测定存在一定“模糊性”，但作者认为不影响结论 [Mogi 1971, pp. 10-13]。  
- 延性度关系中引用了经验常数 c，其物理意义及是否适用于所有岩石尚不明确。  
- 实验中忽略了加载顺序的影响，且未监测孔隙流体压力，不能直接用于含流体情况。

## Assumptions / Conditions

- 所有测试岩石均视为均质各向同性。  
- 压缩应力取正，主应力符号 σ₁ > σ₂ > σ₃。  
- 断裂与屈服准则建立在“应力状态唯一决定破坏”（其他因素如速率、温度、尺寸固定）的假设下 [Mogi 1971, pp. 7-10]。  
- 八面体剪应力 τ_oct 的临界值随有效平均压力单调递增，其函数形式可依据常规三轴实验 (σ₂=σ₃) 确定 [Mogi 1971, pp. 7-10]。  
- 延性度定义为断裂前永久轴向应变，未考虑体积膨胀等其他延性度量。  
- 对地幔的推论隐含认为岩石力学行为的基本函数形式在更高 PT 下仍保持，但缺乏实验依据。

## Key Variables / Parameters

- σ₁, σ₂, σ₃：最大、中间、最小主应力（压缩为正，σ₁>σ₂>σ₃）。  
- 八面体剪应力 τ_oct：τ_oct = (1/3)[(σ₁-σ₂)² + (σ₂-σ₃)² + (σ₃-σ₁)²]^½。  
- 有效平均压力（破裂）：(σ₁+σ₃)/2。  
- 有效平均压力（屈服）：(σ₁+σ₂+σ₃)/3。  
- 延性度 εᵖ：断裂前永久轴向应变 [Mogi 1971, pp. 6-7]。  
- 经验常数 c：表示 σ₃ 效应与 σ₂ 效应对延性度影响的权重比，因岩石而异。

## Reusable Claims

1. 对于致密结晶岩在室温、中低围压 (≤5 kb) 和恒定应变率下，破裂的应力条件可用广义 von Mises 准则：τ_oct = f₁((σ₁+σ₃)/2)。其中 f₁ 是由常规三轴实验 (σ₂=σ₃) 确定的单调递增函数。该准则不显含 σ₂，τ_oct 与 (σ₁+σ₃)/2 的关系独立于 σ₂ 值。

2. 类似条件下，岩石开始塑性屈服的应力条件可用 τ_oct = f₂((σ₁+σ₂+σ₃)/3) 描述，f₂ 单调递增。当 f₂ 为常数时，退化为经典 von Mises 准则（如 Solenhofen 石灰岩）。该准则表明，即使在 τ_oct 增加时屈服仍可延迟，只要平均应力同步增高。

3. 延性度 εᵖ 可由 εᵖ = f₃(cσ₃ - σ₂) 表征，f₃ 单调递增。常数 c 通过实验标定，例如 Dunham 白云岩中 c ≈ 3，表明 σ₂ 对脆性化的贡献约为 σ₃ 的 1/3。这可用于预测不同构造应力状态（σ₂ 相对 σ₃ 的大小）下的脆性程度。

4. 在三轴应力条件下，剪切破裂面平行于 σ₂ 方向，因此破裂面上的有效正应力为 (σ₁+σ₃)/2，而屈服不沿特定宏观平面发生，故采用三维平均应力 (σ₁+σ₂+σ₃)/3。这一物理图像支持了两种准则中平均应力选取的差异。

5. 综合以上，在深部地质体中，若 σ₂ 相对较高，则延性度降低，可能导致原本预期为韧性流动的区域发生脆性破裂（地震），这对理解俯冲带深部地震活动性具有潜在意义，但需考虑高温等因素的修正。

## Connections To Other Work

- 与 von Kármán (1911)、Robertson (1955)、Handin and Hager (1957)、Griggs et al. (1960)、Heard (1960)、Brace (1964) 等通过常规三轴 (σ₂=σ₃) 获得的强度-围压关系一致，本文在此基础上补充了 σ₂ 的作用 [Mogi 1971, pp. 1-2, 6-7]。  
- 与前人尝试实现真三轴应力状态的方法对比：Böker (1915) 的压缩-扭转、Handin et al. (1960) 的空心圆柱压扭、Sakurai and Serata (1967) 等端面润滑方案均存在应力不均匀或摩擦误差，本研究的流体围压 + 独立活塞方案有效的克服了这些不足 [Mogi 1971, pp. 2-3]。  
- Hojem and Cook (1968) 的铜扁平千斤顶方法得到的初步数据与本文新准则大体一致，但压力范围和精度有限 [Mogi 1971, pp. 2-3, 7-10]。  
- 经典 von Mises 屈服准则（τ_oct = const）被视为本文广义准则的一个特例，适用于屈服应力不依赖于压应力的材料 [Mogi 1971, pp. 10-13]。  
- 与 Coulomb-Mohr、Griffith 等传统破裂准则的比较：原有准则未考虑 σ₂ 效应，无法描述本实验观察到的 σ₂ 对强度的显著影响 [Mogi 1971, pp. 7-10, Table 1]。

## Open Questions

- 广义 von Mises 准则在高温、高应变率以及富含孔隙流体的条件下是否依然成立？  
- σ₂ 对脆-延转换的定量影响，尤其是与温度、压力的耦合关系尚需系统研究。  
- 延性度经验关系式中的常数 c 与岩石微结构（如矿物颗粒大小、孔隙度、裂纹密度）有何内在联系？  
- 在地幔温压条件下，橄榄岩等典型地幔岩石是否遵循类似的 τ_oct-f₁((σ₁+σ₃)/2) 破裂准则？若能，如何由高温高压实验标定 f₁ 函数？  
- 实验中 σ₂ 方向对破裂面取向的控制机制，当 σ₂ 与 σ₃ 非常接近时（如接近常规三轴状态），准则是否仍精确？  
- 文中提到 Hojem and Cook 数据有限，是否能有更多独立验证来确立新准则的普适性？
