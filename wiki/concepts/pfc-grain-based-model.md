# PFC 颗粒流模型

## Definition

利用颗粒流方法将岩石抽象为粘结颗粒集合，通过细观参数表征矿物接触和热力行为，用于模拟热开裂等过程。
## Aliases

- PFC
- Particle Flow Code
- parallel bond model
- PFC颗粒流模型
## Boundary

二维模型未考虑三维效应；矿物组分简化，模型参数需多目标优化标定。
