# 岩心直径变形分析 (DCDA)

## Definition

通过高精度测量岩心应力释放后的直径椭圆度，结合弹性理论反演钻孔平面内的主应力差和方向的方法。
## Aliases

- DCDA
- core diametrical deformation
- 岩心变形法
## Boundary

要求岩石近似线弹性各向同性；钻孔轴需近似平行于一个主应力。
