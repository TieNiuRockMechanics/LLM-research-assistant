# 神经网络参数反演

## Definition

利用训练好的 BP 或 CC 神经网络从二维裂隙统计参数反演三维 DFN 模型参数。
## Aliases

- neural network parameter inversion
- NN inversion
## Boundary

反演精度依赖于训练数据库的代表性；仅通过单条平硐数据验证。
