# 软数据与硬数据

## Definition

硬数据指可直接精度测量的数据（如井点厚度），软数据指定性、概念性或统计性的信息（如相组合、宽厚比统计关系）。
## Aliases

- Soft data vs Hard data
- 软数据与硬数据
## Boundary

硬数据用于条件化模型，软数据用于构建概念模型；错误使用软数据可能导致建模错误。
