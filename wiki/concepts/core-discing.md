# 岩芯饼化

## Definition

在高地应力地区钻探取芯时，岩芯自发破裂成饼状或盘状的现象，是应力释放导致的拉-剪复合破坏。
## Aliases

- Core discing
- Core Disking
- 岩饼
## Boundary

主要发生在高应力、较完整结晶岩中；饼化程度与主应力大小和差应力相关。
