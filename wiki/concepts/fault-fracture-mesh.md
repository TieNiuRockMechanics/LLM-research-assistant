# 断层-裂缝网格

## Definition

由剪切断层、张裂缝和混合型裂缝相互交织的三维网络，在近最小主应力方向的高流体压力下可自组织形成高渗透通道。
## Aliases

- fault-fracture mesh
- Hill mesh
## Boundary

需要高流体压力（P_f ≈ σ₃）和非均质岩体；网格沿 σ₂ 方向优先连通。
