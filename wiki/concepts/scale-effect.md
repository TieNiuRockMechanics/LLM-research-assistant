# 尺寸/尺度效应

## Definition

岩石力学性质（强度、模量）和节理粗糙度（JRC）随试样或块体尺寸增大而降低的现象。
## Aliases

- scale effect
- 尺度效应
- size effect
## Boundary

源于非均匀性；JRC 的经典经验尺度校正式为 JRC_n = JRC_0 (L_n/L_0)^{-0.02 JRC_0}，但不能无限外推。
